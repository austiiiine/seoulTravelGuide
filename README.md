# **【iOS SwiftUI 手機應用程式設計入門 #04】電子書 App**

## 簡介

這次的 App 主要紀錄了自己在首爾想吃或曾吃過的美食，以及一些想去逛、去玩的景點。

[pics]

## 資訊架構

App 分為三大主題頁籤：「美食地圖」、「必玩必買」和「已收藏」，各自下方又根據地區或景點類型分類。選擇一個分類進入後，可以查看該分類底下的景點清單，再點選任一景點查看詳細資訊與相片。

## 功能實作說明

- 自訂的資料型別為 Spot，包含景點名稱、營業時間、地址、交通方式及照片集等屬性。
- 以 TabView 搭配 Tab 實作三大主題頁籤：「美食地圖」、「必玩必買」和「已收藏」。
- 主題頁籤中依據景點地區或類型進行分類，使用自訂的CategoryRow 實作「美食地圖」和「必玩必買」這兩個 Tab 內的分類列表。
- 每個分類中的景點列表由自訂的 SpotRow 搭配 List 完成。
- 以自訂的 SpotDetail 呈現景點詳細頁面，並使用 TabView 和 Page 做出橫向滑動相片的效果。
